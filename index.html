Po utworzeniu kodu ssh trzeba go jeszcze przeniesc na stronę GitHub gdzie zostanie on wklejony w ustawienia by mozna było swobodnie przenosić sobie dane.
Aby skopiować ten plik trzeba użyć komendy cat, która wyświetli zawartość pliku w terminalu, a następnie link /c/Users/Plaza/.ssh/id_rsa.pub, który został wygenerowany przez nasz ssh-key gen.
